DragonDrop

This Block-based educational IDE was created to make learning programming more accessable. It was developed as part of a paid internship with the DigiPen Research and Development department and was based on popular frameworks and tools including Blockly, NodeJS, and Electron. This software is used during summer camps and classes at DigiPen!

Bittorio

A 3D puzzle fighter made in Unity as an academic project at DigiPen. This game went to PAX, was nominated for IGF, and is available for download! I Worked on Tools, UI, website/marketing, and did network prototyping using SocketIO and a custom NGINX and Node.js server. See more on the game's website!
