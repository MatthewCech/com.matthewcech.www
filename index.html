DragonDrop

This Block-based educational IDE was created to make learning programming more accessable. ASCIIPlayer

Lightweight cross-platform music player that runs on the command line. The Console visualization uses a console drawing library I wrote to efficiently display rapidly updating characters with similarities between frames, and the audio playback and analysis is done with FMOD. Check it out on asciiplayer.com, or thumb through the source that's available on GitHub!
